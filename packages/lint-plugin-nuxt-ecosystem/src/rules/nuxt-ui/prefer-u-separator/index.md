# `@nustack/nuxt-ui/prefer-u-separator`

Prefer `USeparator` over a raw `hr` element for consistent orientation, color, size, and
optional label handling.

```vue
<!-- Incorrect -->
<hr>

<!-- Correct -->
<USeparator />
```

Use `data-raw` when native markup is intentional.
